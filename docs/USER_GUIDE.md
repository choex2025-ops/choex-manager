## 快速开始

### 启动服务

首先确认后端服务已启动，然后：

```bash
# 启动前端（如果没有启动）
cd choex-web
npm run dev
```

打开浏览器访问 **http://localhost:5173**

### 系统要求

- 现代浏览器（Chrome/Firefox/Safari/Edge）
- 后端服务运行在 localhost:8080

## 注册与登录

### 注册新账号

1. 打开 http://localhost:5173，进入登录页
2. 点击底部 **"立即注册"** 切换到注册模式
3. 填写：
   - **用户名**：至少 2 个字符
   - **邮箱**：用于登录和身份识别
   - **密码**：至少 6 位
4. 点击 **"注册"** 按钮

注册成功后自动登录并跳转到工作台。

### 登录已有账号

1. 在登录页填写邮箱和密码
2. 点击 **"登录"**

### 退出登录

点击左侧栏底部的 ⚙ 设置图标（功能开发中），或清除浏览器 localStorage 后刷新。

## 工作台布局

登录后进入工作台，页面分为四个区域：

```
┌──────┬──────────────────────────────┬───────────┐
│      │                              │  Agent    │
│ 左   │         中间区域              │  对话面板  │
│ 侧   │        (动态内容切换)          │           │
│ 应   │                              │  🧠记忆   │
│ 用   │                              │  ⚙ 设置   │
│ 栏   │                              │           │
│      │                              │  对话     │
│      │                              │  记录     │
├──────┴──────────────────────────────┴───────────┤
│              底部 AI 输入栏                       │
└────────────────────────────────────────────────┘
```

**左侧应用栏**（64px 宽）：竖直排列的应用图标，点击切换中间区域内容。
- 📅 日程管理
- 💳 记账
- 🔐 密码簿
- 🌐 浏览器
- ⚙ 系统设置（待开发）

**中间区域**：默认显示欢迎页，点击左侧应用后切换为对应功能界面。再次点击同一应用可以取消选中，回到欢迎页。

**右侧 Agent 面板**（320px）：显示与生活管家的对话记录，顶部有 🧠 记忆管理和 ⚙ 更多设置入口。

**底部输入栏**：全局 AI 输入框，可随时输入指令或消息发送给 Agent。

## 日程管理

点击左侧 📅 进入日程管理。

### 查看日程

所有已创建的日程以列表形式展示，每条显示：
- 标题
- 时间范围（开始 ~ 结束）
- 地点（如有）
- 左侧色条（颜色标签）

列表按时间升序排列。

### 创建日程

1. 点击右上角 **"+ 新建日程"** 按钮
2. 填写表单：
   - **标题**（必填）：日程名称
   - **地点**：会议地点或位置
   - **开始时间 / 结束时间**：点击选择日期和时间
   - **描述**：日程详情
   - **颜色**：标签颜色（用于区分不同类型）
3. 点击 **"保存"**

### 删除日程

点击日程条右侧的 ✕ 按钮即可删除。

### 示例

| 操作 | 步骤 |
|---|---|
| 创建周会 | 新建 → 标题"周会" → 时间：周一10:00~11:00 → 地点"会议室A" → 保存 |
| 标记重要日程 | 创建时选择红色，列表中红条醒目显示 |
| 查看一天的安排 | 浏览列表，时间已按顺序排列 |

## 记账

点击左侧 💳 进入记账。

### 查看概览

页面顶部显示**月度统计栏**：

| 指标 | 说明 |
|---|---|
| 本月收入 | 绿色显示，当月所有收入总和 |
| 本月支出 | 红色显示，当月所有支出总和 |
| 结余 | 收入 - 支出 |

### 记一笔

1. 点击 **"+ 记一笔"** 按钮
2. 选择类型：
   - **支出**：花出去的钱
   - **收入**：进账的钱
3. 选择**分类**：餐饮 / 交通 / 购物 / 娱乐 / 其他
4. 输入**金额**
5. 选择**日期**（默认为今天）
6. 填写**备注**（可选）
7. 点击 **"保存"**

### 查看记录

所有记录以列表展示，每条显示：
- 金额（绿色=收入，红色=支出）
- 分类标签
- 备注
- 日期

### 删除记录

点击记录右侧的 ✕ 按钮。

### 使用建议

| 场景 | 操作 |
|---|---|
| 记录午餐 | 支出 → 餐饮 → 30元 → "和同事吃的拉面" |
| 记录工资 | 收入 → 15000元 → "6月工资" |
| 查看本月餐饮花费 | 查看月度统计栏，所有餐饮类支出自动汇总 |
| 记昨天的账 | 点击日期选择器，选昨天即可 |

## 密码簿

点击左侧 🔐 进入密码簿。

### 安全说明

- 所有密码使用 **AES-256-GCM** 加密存储
- 密码明文不会出现在列表中，需要**点击查看**
- 仅本人可查看（user_id 隔离）

### 新增密码

1. 点击 **"+ 新增"** 按钮
2. 填写：
   - **标题**：如"GitHub"或"公司邮箱"
   - **网址**（可选）：对应网站
   - **用户名**：账号
   - **密码**：密码原文（保存后加密存储）
   - **分类**（可选）：便于分组管理
3. 点击 **"保存"**

### 查看密码

1. 在列表中找到对应条目
2. 点击 👁 眼睛图标
3. 密码明文显示（绿色背景）
4. 再次点击 🙈 隐藏

### 删除密码

点击条目右侧的 ✕ 按钮。

### 使用建议

| 分类 | 示例条目 |
|---|---|
| 工作 | 公司邮箱、VPN、内部系统 |
| 开发 | GitHub、GitLab、云服务器 |
| 社交 | 微信公众平台、微博 |
| 财务 | 网银、支付宝 |

## 嵌入式浏览器

点击左侧 🌐 进入浏览器。

### 使用方式

1. 在地址栏输入网址
2. 点击 **"前往"** 或按 Enter
3. 网页在中间区域显示

### 特点

- **无需切换浏览器**：在应用中直接访问网页
- **后端代理**：部分禁止 iframe 的网站，通过后端代理加载
- **配合 Agent**：让 Agent 帮你打开指定页面（未来功能）

### 适用场景

- 查看技术文档
- 访问内部系统
- 调试页面（如本项目的设计稿在 `http://localhost:60929`）

## Agent 对话

Agent 是你的个人生活管家，可以**对话聊天**，也可以**直接操作应用**。

### 基础对话

在底部输入框输入任何内容，按 Enter 或点击"发送"，Agent 会流式回复。

```
你：你好
Agent：你好！我是你的生活管家。需要什么帮助吗？
```

回复内容逐字显示在右侧面板中，体验流畅。

### 工具调用（Tool Calling）

Agent 可以**操作你的应用数据**。试试这些指令：

#### 记账相关

| 你对 Agent 说 | Agent 会做什么 |
|---|---|
| "帮我记一笔午餐30元" | 自动创建一条餐饮支出记录 |
| "记一笔收入15000元" | 创建一条收入记录 |
| "我今天花了多少钱" | 返回今日账单列表 |
| "我本月餐饮花了多少" | 返回月度统计 |

#### 日程相关

| 你对 Agent 说 | Agent 会做什么 |
|---|---|
| "帮我创建一个明天下午3点的会议" | 自动创建日程 |
| "我明天有什么安排" | 列出明天所有日程 |
| "显示我本周的所有日程" | 返回本周日程列表 |

#### 工作原理

当你发出一条指令后：
1. Agent 判断是否需要调用工具
2. 如需操作，Agent 自动执行（创建日程/记账等）
3. 操作完成后，Agent 用自然语言告诉你结果

你可以在右侧对话面板看到完整的交互过程。

## 记忆管理

记忆是 Agent 的"人格文件"——定义 Agent 的性格、已知信息、行为偏好。

### 打开记忆管理

点击右侧 Agent 面板顶部的 🧠 图标，中间区域打开记忆列表。

### 记忆列表

每条记忆以卡片形式展示：

```
┌─────────────────────────────────────────────┐
│ ☀  日常助手                          [开关] │
│    最后修改: 2026/6/11 14:30         [✕]   │
└─────────────────────────────────────────────┘
│ 💼 工作模式                          [开关] │
└─────────────────────────────────────────────┘
│ 🎮 游戏搭子                          [开关] │
└─────────────────────────────────────────────┘
```

- **激活态记忆**：蓝色边框 + "使用中"标签 + 蓝色滑动开关
- **未激活记忆**：灰色背景 + 灰色开关

### 切换记忆（滑动开关）

- 点击某条记忆的**滑动开关** → 该记忆激活，Agent 立即使用新人格
- **同一时刻只能激活一条记忆**：激活新记忆时，原来的自动关闭
- 切换后，Agent 对话面板会出现提示："已切换到XX记忆"

### 新建记忆

1. 在列表顶部输入框中输入新记忆名称（如"工作模式"）
2. 点击 **"+ 新建"**
3. 新记忆自动创建，点击即可编辑

### 编辑记忆

点击某条记忆卡片 → 进入编辑页。

编辑页有三个版本标签：

| 版本 | 用途 | 可编辑 |
|---|---|---|
| **当前版本** | Agent 实际使用的版本 | 是 |
| **备份版本** | 上次保存前的版本（防误操作） | 否（只读） |
| **自定义版本** | 手动保存的特别版本 | 是 |

#### 编辑当前版本

1. 在文本编辑器中修改内容
2. 点击 **"保存修改"**
3. 旧版本自动移入备份版本槽位

#### 从备份恢复

点击 **"从备份恢复"** → 备份版本的内容恢复到当前版本。

#### 存入自定义版本

编辑内容后，点击 **"存入自定义版"** → 保存到自定义版本槽位（不受自动备份影响）。

### 记忆内容示例

**日常助手（温和、细心）：**

```
你是 Choex 的个人生活管家。
性格：温和、细心、有规划。

关于用户：
- 偏好早上处理重要事务
- 工作日午餐预算 30-50 元
- 周五下午做下周计划
```

**工作模式（简洁、高效）：**

```
你是工作助手。回复简洁，只说要点。
不闲聊，不主动建议。
专注：日程安排、工作提醒、时间管理。
```

### 使用建议

1. **创建多个记忆文件**，匹配不同场景（工作/生活/休闲）
2. **定期更新记忆**：添加新的偏好或习惯
3. **利用备份**：大胆修改，改错了可以随时恢复
4. **自定义版本**：保存一个"黄金版本"作为人格模板

## 常见问题

### Q: 忘记密码怎么办？
A: 当前版本暂不支持密码重置。请联系管理员在数据库中修改密码哈希。

### Q: 数据安全吗？
A: 密码簿中的密码使用 AES-256-GCM 加密存储。所有数据通过 user_id 隔离，其他用户无法访问你的数据。

### Q: Agent 能访问我的密码簿吗？
A: 当前版本的 Agent 暂不支持直接读写密码簿（仅日程和记账）。未来会加入更安全的鉴权机制。

### Q: 为什么有些网站在浏览器中打不开？
A: 部分网站设置了 X-Frame-Options 头禁止 iframe 嵌入。后端代理会自动处理大部分情况，但仍有少量网站无法加载。

### Q: 如何反馈问题？
A: 请在 GitHub 仓库提交 Issue：
- https://github.com/choex2025-ops/choex-server/issues
- https://github.com/choex2025-ops/choex-web/issues
